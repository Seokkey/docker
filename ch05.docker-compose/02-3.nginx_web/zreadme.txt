===============================================================================================
nginx+web with docker compose 구성에서 composetest의 사례와 같이 실시간으로 app.py가 적용되도록 구성한 사례
===============================================================================================
1) 01-1 사례에서의 app 디렉토리 하위에 있는 모든 파일을 프로젝트 루트 디렉토리로 이동하여 composetest의 구성과 동일하게 함
2) 웹 어플리케이션 서비스는 composetest 사례의 Dockerfile을 그대로 사용하고
3) app.py는 01-1 사례의 app.py를 수정하여 사용
4) docker-compose.yml 파일은 composetest의 사례와 01-1 사례를 혼합하여 필요한 부분으로 수정.
--> composetest 사례는 redis를 사용하는 반면 01-1 사례는 nginx를 사용
5) docker-compose up 명령을 실행하면 __pycache__ 폴더가 생성되고
--> 결과적으로 컨테이너가 실행 중에도 app.py가 실시간으로 갱신되어 적용된다
